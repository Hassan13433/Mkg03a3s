❓ Hilfe

Laden Sie Ollama unter ollama.ai herunter und installieren Sie es. Starten Sie dann einen Modell-Server mit:

Keine Modelle vorhanden?

Laden Sie ein Modell herunter mit:

ollama pull mistral
